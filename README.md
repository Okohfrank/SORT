# SortAI — Intelligent Waste Sorting System

> AI-powered waste classification combining Computer Vision, 
> NIR sensing, and dynamic pneumatic control.

🔗 **Live Demo:** [sortai.netlify.app](https://sortai.netlify.app)

---

## Overview

SortAI addresses one of Africa's most pressing environmental 
challenges, mixed waste mismanagement. Unlike conventional 
sorting systems that apply fixed separation parameters, SortAI 
dynamically adjusts air-jet force in real time based on waste 
characteristics detected through Computer Vision and Near 
Infrared (NIR) sensing — improving sorting accuracy across 
mixed plastic and organic waste streams.

---

## Key Innovation

Traditional waste sorting systems apply a **fixed** air-jet 
force to separate waste — leading to misclassification when 
waste properties vary. SortAI solves this by:

- Detecting waste type in real time using Computer Vision
- Characterizing material properties via NIR sensing
- **Dynamically adjusting pneumatic force** based on detected
  characteristics
- Logging and visualizing sorting data on an AI-powered dashboard

